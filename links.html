Resources and more

During the process on improving my coding skills, I'm gathering all kinds of resources for myself that might be useful to others. I'll be sharing them here.

Setup

  • VSCode. I've used both Atom and Sublime Text before and though I like both, this is my favorite.
  • Roost Laptop Stand: very lightweight, easy to take with you anywhere.
  • Microsoft Sculpt Egonomic Keyboard (works well with a MacBook as well)
  • Bash shortcuts: I use #3 and #8 from this list for easier terminal navigation on Mac.

Courses

These are all course I've taken myself/am currently taken.
Course Platform Notes
CS50: Web Programming with Python and Javascript Edx (free) Currently taking this course myself.
Vue JS 2 - The Complete Guide Udemy (~$10, wait till it goes on sale) This is a great overview of Vue.js, taking you from creating simple to more advanced projects and web apps. You do need basic knowlegde of HTML, CSS and Javascript to take this course.
CS50: Introduction to Computer Science Edx (free) I took a minor at the University of Amsterdam that used a lot of CS50's coursework. If you're looking for a good foundation in CS, problem solving, debugging and general programming skills, I'd highly recommend it.
